# Kakuma Food Distribution Analysis

## Business Understanding
Refugee households in Kakuma often face inconsistencies in food distribution, which can lead to shortages, inefficiencies, and unequal resource allocation. The purpose of this project is to analyze historical distribution data to identify patterns, bottlenecks, and areas for improvement. By providing actionable insights, this analysis supports NGOs, aid agencies, and camp administrators in making data-driven decisions to enhance fairness, efficiency, and transparency in food distribution.

## Project Overview
This project involves cleaning, analyzing, and visualizing food distribution data to uncover trends and generate insights that can improve operational planning and reporting.

## Features
- Data cleaning and preprocessing of raw distribution datasets
- Exploratory Data Analysis (EDA) to detect trends and bottlenecks
- Visual reporting for stakeholders
- Recommendations to optimize resource allocation

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Excel
- Power BI (for interactive dashboards)

## Key Achievements
- Identified inefficiencies and patterns in food distribution
- Generated actionable insights for better decision-making
- Enhanced reporting accuracy and clarity for stakeholders
